须通过 HTTP 打开（不要用 file:// 双击 HTML）。改代码后请强制刷新（Ctrl+F5）。
启动本地服务（二选一）
  1. Docker 全栈：项目根目录 cd frontend && npm run docker:go-live → 打开 http://127.0.0.1:8000/static/HengAI_本地UI预览台.html
  2. 仅静态：cd frontend 后执行 python -m http.server 8080 → 打开 http://127.0.0.1:8080/HengAI_本地UI预览台.html
