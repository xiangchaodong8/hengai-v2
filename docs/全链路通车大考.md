# HengAI V2 · 全链路通车大考（人工验收清单）

> 代码侧批次 1–8 已完成静态/契约校验。本文档用于 **上线前最后一次人工走查**。  
> 建议：Chrome 无痕窗口 + `Ctrl+F5` 强刷，后端 `docker compose up` 或本地 `uvicorn`。

## 0. 发车前自动化（约 2 分钟）

### 本地代码校验

```powershell
cd frontend
npm run preflight
```

### Docker 一键发车（批次 9 · 推荐）

```powershell
cd frontend
npm run docker:go-live
```

含：preflight → `docker compose up -d --build` → `/api/health` 等待 → 运行栈烟测。  
详见 **[Docker发车指南.md](./Docker发车指南.md)**。

容器已启动、仅补烟测：

```powershell
npm run smoke:live
```

全绿后再做下列人工步骤。若需含浏览器烟测：

```powershell
npm run preflight:full
```

---

## A. 首页对话 → SSE 共振

| # | 操作 | 预期 |
|---|------|------|
| A1 | 打开 `index.html`，登录（或访客） | 无控制台红色报错；`AppState` 已加载 |
| A2 | 发送一条含企业数据的消息（如年用电量） | 出现专家回复流 |
| A3 | 打开 DevTools → Network → `chat` SSE | 首帧为 `event: actions_taken`，`data` 含 **`updatedState`**（非 `updated_state`） |
| A4 | 查看 `updatedState.user` | 含 `tier_code`、`regDate`；有数据时含 `regLabel` |
| A5 | 查看 `updatedState.metrics` | 有 CBAM/报告后 `riskExposureEur`、`roiMultiple` 为数字，非空壳 |

---

## B. 全域中心 · 母页数字

| # | 操作 | 预期 |
|---|------|------|
| B1 | 点击右上角 **`[全域中心]`** | 左上角阶段/企业名与 AppState 一致 |
| B2 | 查看 ROI / 碳税相关 `.dyn-rep-*` | 有报告后 **不是** 长期停留在 `—` 或「待测算」 |
| B3 | 查看用户注册时间 / 会员档 | 与账号实际一致；访客显示「访客」 |
| B4 | 切换 Phase2/3 Tab（若已解锁） | iframe 内模块加载，无 404 |

---

## C. 企业数字档案 → 管道广播

| # | 操作 | 预期 |
|---|------|------|
| C1 | 全域中心 → 左侧 **`[企业数字档案]`** | iframe 内企业名、行业与保存一致 |
| C2 | 修改企业名并保存 | 成功 toast；GM 动效（若已 Pro） |
| C3 | 回到 **`[全域诊断报告]`** iframe | 企业名已同步，无需手动刷新 |
| C4 | 回到首页聊天区 | 下一轮对话 System 摘要含新企业名 |

---

## D. CBAM 测算 → 财务数字

| # | 操作 | 预期 |
|---|------|------|
| D1 | 全域中心 → **`[CBAM 测算工具]`** | 可完成粗测/保存 |
| D2 | 保存后看母页 ROI、碳税 | `computeRepFinancials` 驱动显示，如 `1 : x.x` |
| D3 | **`[决策层呈送包生成器]`** | 文案中风险/ROI 与母页同量级 |
| D4 | **`[供应链协同]`** | 覆盖率 % 与 `metrics.supplyChainCoverage` 一致 |

---

## E. 子模块抽样（12+ iframe）

至少抽 3 个模块确认 **管道一致**：

- `HengAI_全域诊断报告.html`
- `HengAI_供应链协同.html`
- `HengAI_GM_Wallet.html`

| # | 检查项 | 预期 |
|---|--------|------|
| E1 | 页内无「王磊」「汽配制造厂」等幽灵样例 | 已通过 CI，再目视确认 |
| E2 | 会员档 / 注册时间 | 与母页一致 |
| E3 | 碳税/ROI 展示 | 与母页同数据源，非各写一套假数 |

---

## F. 新用户 / 访客轨

| # | 操作 | 预期 |
|---|------|------|
| F1 | 退出登录，清 Token，强刷 | `tier_code` = `GUEST` |
| F2 | `/api/v1/hub/overview` 无 Token | 200 + 访客 DNA，不 401 |
| F3 | 第 6 轮对话或下载报表（Guest） | Paywall 弹层（若产品开关已接） |

---

## G. 回归红线（一票否决）

- [ ] 任意核心页 **长期** 显示「待测算」且已有 CBAM 报告
- [ ] 母页与 iframe **ROI/碳税数字矛盾**（差一个数量级）
- [ ] 聊天 SSE **没有** `actions_taken` 或 `updatedState` 为空对象
- [ ] 控制台持续 `patchAppState` / 管道 **死循环** 报错

```powershell
cd frontend
npm run sync:dist
```

确认 `static_dist/AppState.js` 与 `frontend/AppState.js` 一致（`preflight` 已含此项）。
