### 4.3 LOGO 使用原则（供设计/印刷）

| 规则 | 说明 |
|------|------|
| 主标识 | **Co2Lion** 拉丁字标 + 中文「方舟数字科技」组合；BP 封面以字标为主，不强行加图形 unless 品牌部提供矢量稿 |
| 产品标识 | **HengAI**、**CL-IVC**、**CL-GTS**、**方舟魔盒** 在四层架构图中分级出现，不与公司主 LOGO 抢视觉中心 |
| 色彩 | 绿色贸易基础设施：主绿（合规/增长）+ 金色（主权/价值）；与 BP 内「Lv.4 / 确权 / 绿印」语义一致 |
| 禁用 | 不得将「西值东送」与**财政补贴比例**、**政府背书承诺**绑在同一视觉块（合规红线） |
| 保密 | 封面保留 v2 警示语：⚠ 本文件含有商业机密，仅供被授权接收方阅读 |
