# 商业引导话术 v2 · 首页 → 实证中心 → 共振触发 → 全域中心

> **版本**：2026-06 · 对齐《三界定位 v2》《SYNC_CONTRACT_v2》  
> **用途**：`index.html` 对话首页 · 聊天 Widget · 供应链协同 · Hub 空态/卡片 · Toast/引导链接

## 1. 角色与目标

| 角色 | 首要焦虑 | 商业目标 |
|------|----------|----------|
| **下游 SME** | 默认因子罚息、上游不配合 | 建档 → 施压 → 共振触发 → 等待实证 → Pull |
| **原厂（大厂）** | 排斥硬件、要先看清差距 | 精算芯付费或接受共振触发 → CL-GTCID → 魔盒 |
| **访客** | 不知道敞口多大 | Widget 粗测 → Paywall → 进 Hub |
